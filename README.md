# thrive
Hardcore bullet hell game with RPG elements.

### Steps To Run the Project Locally
1. clone repository
1. In the command line:
    > `nvm use`
    1. If you need a new version of node, then in the command line:
    > `nvm install`

    > `nvm use`
1. In the command line:
    > `npm install`
1. In the command line:
    > `npm start`
